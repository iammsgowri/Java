# Constructor

## Constructor Definition

```java
"CONSTRUCTOR" is a "SPECIAL METHOD" used to "INITIALIZE OBJECTS"
```

```java
"CONSTRUCTOR" should be created using "SAME NAME" as "CLASSNAME"
```

```java
"CONSTRUCTOR" doesn't have any "RETURN TYPE" not even "VOID"
```

```java
"CONSTRUCTOR" gets "CALLED" at the "TIME" of "OBJECT CREATION" itself
```

## Constructor Syntax

```java
public class ClassName {

	accessmodifier ClassName() {
	
	}
	
}
```

# Constructor Types

# Default Constructor

## Default Constructor Definition

```java
"DEFAULT CONSTRUCTOR" doesn't take any "PARAMETERS"
```

```java
If there is "NO DEFAULT CONSTRUCTOR", inside the class, "JVM" automatically creates "DEFAULT CONSTRUCTOR"
```

```java
public class ClassName {

	// no default constructor
	
	accessmodifier ClassName() {
	
	}
}
```

```java
"DEFAULT CONSTRUCTORS" gets "CALLED" at the "TIME" of "OBJECT CREATION" itself
```

```java
ClassName objectReferenceVariable = new ClassName();
```

## Default Constructor Syntax

```java
public class ClassName {

	accessmodifier ClassName() {
	
	}
	
}
```
# Parameterized Constructor

## Parameterized Constructor Definition

```java
"PARAMETERIZED CONSTRUCTOR" takes "PARAMETERS"
```

## Parameterized Constructor Syntax

```java
public class ClassName {

	accessmodifier ClassName(paramters) {
	
	}
	
}
```
