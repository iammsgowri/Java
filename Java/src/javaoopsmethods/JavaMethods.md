# Java Methods

## Method Definition

```java
"METHOD" is a "BLOCK" of "CODE" used to perform "SPECIFIC TASK" 
```

## Method Syntax

```java
accessmodifer returntype methodName(parameters){
}
```

## Method Return Type

```java
"METHOD" can have "RETURN TYPE" as "PRIMITIVE DATATYPE" or "NON PRIMITIVE DATATYPE"
```

```java
public String enteredEmail(String email) {
		return email;
	}
```

```java
"METHOD" can have "RETURN TYPE" as "VOID" which doesn't have "ANYTHING" to "RETURN"
```

```java
accessmodifer void methodName(){
}
```

# Method Types

## Non-Parameterized Method

```java
"NON-PARAMETERIZED" method doesn't take any "PARAMETERS"
```

```java
"NON-PARAMETERIZED" method may or may not have "RETURNTYPE"
```

## Non-Parameterized Method Syntax

```java
accessmodifer void methodName(){
}
```

```java
accessmodifer returntype methodName(){
	return ;
}
```

## Parameterized Method

```java
"PARAMETERIZED" method take "PARAMETERS"
```

```java
"PARAMETERIZED" method may or may not have "RETURNTYPE"
```

## Parameterized Method Syntax

```java
accessmodifer returntype methodName(parameters){
}
```

```java
accessmodifer methodName(parameters){
	return ;
}
```

## Method Invocation / Call

```java
Once we declared "METHODS", it will "NOT" get "EXECUTED" on its "OWN"
```

```java
We need to "EXPLICITY CALL" method or "INVOKE" method
```
